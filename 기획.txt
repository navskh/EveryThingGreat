- 기능
  0. 로그인/회원가입
    1) 회원가입
      : 아이디/비밀번호/핸드폰번호
      : 핸드폰 번호, 비밀번호 암호화
    2) 로그인
      : 아이디/비밀번호
      : 아이디 저장 기능
    3) JWT 
      : 글 조회 부터 안되게 막을 것
      (네비게이션 가드? 사용)

  1. 글 조회 
    1) 정렬
      : 날짜순 정렬
      : 장절순 정렬
    2) 페이징
  2. 글 작성
    1) 에디터
    : CK에디터
    : TipTap 에디터
    예쁜건 TipTap이 더 예쁘지 않은가.
    * 기능 중에, () 안에 장절을 입력하면 알아서 끄집어 내주는 기능이 있었으면 좋겠다.

    2) 에디터 메뉴
    : H1, H2, H3
    : 리스트
    : 인용구

    3) 중간 저장
    : Local Storage에 글 Model에 맞게끔 저장해둘 것.
    : 만약에 있으면 저장한 글로 돌아가게끔 (해당 노트북에서만 가져올 수 있도록)

  3. 검색
    : 키워드로, 제목, 내용, 카테고리 모두 검색

  4. 사이드바
    : 카테고리 별로 나올 수 있도록
    : 전체보기
